# WorkPsych Decision Lab

Interactive Industrial/Organizational Psychology case simulator for applied classroom activities.

## Files

- `WPDLindex.html`
- `WPDLstyles.css`
- `WPDLdata.js`
- `WPDLapp.js`
- `WPDLREADME.md`

## Features

- 100 I/O Psychology cases
- Topic, difficulty, and mode filters
- Immediate answer checking with rationales
- Job analysis builder
- Selection system designer
- Training needs simulator
- Appraisal error trainer
- Motivation theory matcher
- Readiness tracker
- Exportable study report
