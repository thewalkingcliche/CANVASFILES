# BoardPrep Psych Lab

Interactive board exam simulator for a combined Psychological Assessment and Industrial/Organizational Psychology course.

## Files

- `BPPLindex.html`
- `BPPLstyles.css`
- `BPPLdata.js`
- `BPPLapp.js`
- `BPPLREADME.md`

## Features

- 100 board-style items
- Psychological Assessment and I/O Psychology domains
- Topic and difficulty filters
- Immediate rationale feedback
- Wrong answer review
- Topic readiness tracker
- Standard score trainer
- Item analysis trainer
- Exportable student study report
