# AbDev BoardPrep Lab

A static browser app for a combined Abnormal Psychology and Developmental Psychology board exam course.

## Files

- `ADPLindex.html`
- `ADPLstyles.css`
- `ADPLdata.js`
- `ADPLapp.js`
- `ADPLREADME.md`

## Features

- 100 board-style multiple-choice items
- Abnormal Psychology mode
- Developmental Psychology mode
- Diagnostic mixed quiz
- Topic and difficulty filters
- Immediate feedback with rationales
- Wrong answer review
- Readiness tracker by topic
- Case reasoning lab
- Concept trainer
- Exportable study report
