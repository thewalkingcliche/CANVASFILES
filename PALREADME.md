# Psych Assessment Lab

Interactive static web app for a Psychological Assessment course.

## Files

- `PALindex.html`
- `PALstyles.css`
- `PALdata.js`
- `PALapp.js`
- `PALREADME.md`

## Features

- Assessment Selector
- Score Interpreter
- Assessment Case Lab
- Ethics and Test Use Checker
- Item Analysis Trainer
- Brief Assessment Report Builder
- Exportable Learning Log

## Deployment

Upload the extracted files to a GitHub Pages repository. Then embed in Canvas using:

```html
<iframe 
  src="https://thewalkingcliche.github.io/CANVASFILES/PALindex.html" 
  width="100%" 
  height="950" 
  style="border: 1px solid #ddd; border-radius: 12px;" 
  loading="lazy"
  allowfullscreen>
</iframe>
```

No API key or backend is required.
