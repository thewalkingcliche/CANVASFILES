# NeuroPsych Explorer Lab

Interactive static web app for a Physiological Psychology course. It uses brain–behavior maps, neurochemical explorers, sleep and stress simulations, case-to-brain mapping, lesion exploration, and an exportable learning log.

## Files

- `NPELindex.html`
- `NPELstyles.css`
- `NPELdata.js`
- `NPELapp.js`
- `NPELREADME.md`
